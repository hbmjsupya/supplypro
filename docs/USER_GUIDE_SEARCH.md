# Warehouse Advanced Search User Guide

## Overview
The Advanced Search feature allows you to filter warehouses based on multiple dimensions, including basic info, stock, and management.

## Features

### 1. Multi-Condition Filtering
- **Basic Info**: Search by Warehouse Name, Code.
- **Region**: Filter by Province/City/District.
- **Status**: Filter by Active/Inactive status.
- **Stock**: Find warehouses containing specific Products (by Name/SKU).
- **Manager**: Find warehouses managed by specific users (by Name/Phone).

### 2. Search History
- The system automatically saves your last 10 search keywords.
- Click on a history tag to quickly re-apply the search.
- Click the "Trash" icon to clear history.

### 3. Saved Search Schemes
- You can save a combination of filters as a "Scheme" (e.g., "North Region - Electronics").
- **Save**: After setting filters, click "Save Scheme", name it, and save.
- **Apply**: Select a scheme from the dropdown to apply all filters instantly.
- **Manage**: Delete obsolete schemes.

## How to Use
1. Click the "Advanced Search" toggle (or Filter icon) on the Warehouse List page.
2. Fill in the desired fields.
3. Click "Search" to filter results.
4. (Optional) Save the criteria for future use.

## Tips
- **Fuzzy Search**: Name fields support partial matches.
- **Performance**: Selecting a specific region improves search speed.
