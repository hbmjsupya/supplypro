# SupplyPro 供应链管理系统 - 操作手册

## 采购管理

### 入库采购单流程
本系统支持快速创建和管理入库采购单。

#### 1. 创建入库采购单
1. 进入「采购管理」-「新增入库采购单」页面。
2. **选择供应商**：在下拉列表中选择供应商。
   - *提示*：系统仅显示状态为“已启用”的供应商。
3. **选择分仓**：选择收货仓库，系统会自动填充仓库地址和联系人信息。
4. **添加商品**：点击“添加商品”按钮，在弹窗中选择商品。
   - *提示*：弹窗仅展示“已上架”的商品。支持通过商品名称或编码搜索。
5. **填写数量**：确认商品后，在列表中输入采购数量。
6. **提交**：点击“提交”按钮。
   - *说明*：提交后，系统自动生成“入库采购”类型的采购单，并同步生成一条状态为“待确认”的入库单。无需额外的审批流程。

#### 2. 取消采购单
对于已创建但尚未进入结算流程的入库采购单，支持取消操作。
1. 进入「采购管理」-「采购单列表」页面。
2. 找到状态为“已确认”且结算状态为“未结算”的入库采购单。
3. 点击操作栏中的“取消”按钮。
4. 在弹出的确认对话框中点击“确认”。
   - *结果*：采购单状态变更为“已取消”，关联的入库单也会同步取消。

### 常见问题
- **Q: 为什么找不到某个供应商？**
  - A: 请确认该供应商是否已被禁用。系统仅允许选择已启用的供应商。
- **Q: 为什么商品列表中没有我要的商品？**
  - A: 请确认商品是否已上架。待选品或已下架的商品无法添加到采购单。
- **Q: 采购单提交后去哪里审批？**
  - A: 入库采购单已移除审批环节，提交即生效，请直接进行入库操作。
